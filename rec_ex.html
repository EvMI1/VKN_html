Задачи с рекурсивными вычислениями


Задача "вывод перевернуйтой строки"

Условие:

Пользователь вводит последовательность символов до символа «*». Вывести последовательность в обратном порядке.

Решение на языке С

        
void print()
{
	char s;
	 scanf("%s", &s);
	if (s != '*')
	print();
	printf("%s", s);
}
